УСТАНОВКА:
1. Создайте любою базу и выполните на ней points.sql
2. Правильное название хоста, базы, имя пользователя и пароль следует вписать в base_defaults.php

СИНТАКСИС

GET <путь к проекту>/points/ - выводит все точки, имеющиеся в базе
GET <путь к проекту>/points/round/?radius=<число>&points=<число>&center=dd.dddddd,dd.dddddd - выводит из базы данных точки, находящиеся вокруг
центра (center) в радиусе не более radius метров, точек будет выведено не больше, чем points. Все три параметра строго обязательны, иначе программа
выдаст сообщение об ошибке. center - пара координат,числа с плавающей точкой, записанные через запятую без пробелов.

DELETE <путь к проекту>/points/<число>/ - удаляет из базы данных точку с id <число>.

POST <путь к проекту>/points/ - добавляет новую точку в базу. Данные берутся из формы. в форме должно быть два поля: name - текст, можно написать что угодно,
и coords - широта и долгота, два числа с плавающей точкой, записанные через запятую без пробелов, например 56.739212,38.860103

POST <путь к проекту>/points/<число>/ - редактирует точку с id <число>. Данные берутся из формы, те же поля и тот же формат, что и в случае создания, но 
не обязательно заполнять оба, можно только одно, это значение и будет отредактировано.
